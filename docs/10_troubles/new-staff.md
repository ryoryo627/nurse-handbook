# 新人が不安定

## まずこれだけ

新人が不安定なときは、まず話を聞きます。
1人で抱え込ませず、支える人を増やします。
患者さんの安全と新人の心の健康、両方を守ります。

## できた状態

- 新人が困っていることがわかっている
- 支える体制ができている
- 新人が「相談していい」と思えている
- 患者さんの安全が守られている

## やり方

### 1. まず話を聞く

| やること | 言うこと |
|----------|----------|
| 声をかける | 「最近どう？」「困ってることある？」 |
| 話を聞く | 「それは大変だったね」 |
| 責めない | 「失敗は誰でもある」 |
| 次を考える | 「どうしたらいいか一緒に考えよう」 |

### 2. 支える体制を作る

| 支える人 | 役割 |
|----------|------|
| プリセプター | 日々の指導、相談相手 |
| 主任・師長 | 全体を見る、必要な調整 |
| 同期 | 悩みを共有、励まし合い |
| 教育担当 | 研修、フォローアップ |

### 3. 具体的な対応

| 状況 | 対応 |
|------|------|
| 業務についていけない | 業務量を調整、ペースを合わせる |
| 人間関係がうまくいかない | 配置を見直す、間に入る |
| 自信がない | 小さな成功体験を作る、認める |
| 体調が悪い | 休ませる、産業医に相談 |

## 例外

- 本人が「辞めたい」と言ったら、まず話を聞いて上司に相談
- 体調不良が続くなら、無理させず休ませる
- 患者さんの安全に関わるなら、業務から外すことも検討

## 相談

- **誰に**：看護部 教育担当
- **いつまでに**：気になったら1週間以内
- **何を言う**：「○○さんが不安定。具体的には○○。対応を相談したい」

## 禁止

- 放置する
- 「最近の若い人は」と片付ける
- 本人のいないところで噂話をする
- 1人で抱え込ませる
