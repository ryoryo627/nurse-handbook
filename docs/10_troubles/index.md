# 困ったとき

## まずこれだけ

困ったときは、このページから入ってください。
よくある困りごとへの対処法がまとまっています。

## よくある困りごと

| 困りごと | ページ |
|----------|--------|
| 人が足りない | [人が足りない](not-enough-people.md) |
| 苦情が来た | [苦情が来た](complaint-came.md) |
| 事故が起きた | [事故が起きた](incident-happened.md) |
| 新人が不安定 | [新人が不安定](new-staff.md) |

## 困りごとの見つけ方

困りごとが見つからないときは：

1. **検索する**：画面上の検索窓にキーワードを入れる
2. **関連ページを見る**：各ページの「関連」リンクをたどる
3. **相談する**：看護部 教育担当に聞く

## 緊急時

- 患者さんの命に関わる → まず対応、あとで報告
- 暴力・暴言がある → その場を離れて、上司を呼ぶ
- 災害が起きた → 病院の災害マニュアルに従う
