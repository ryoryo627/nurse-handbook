# 人が足りない

!!! tip "まずこれだけ"
    人が足りないときは、まず上司に報告します。
    1人で抱え込まず、応援を頼みます。
    患者さんの安全が最優先です。

## このページのゴール

- 上司が状況を把握している
- 応援が来ている、または業務を調整している
- 患者さんの安全が守られている

## やり方

### 1. すぐにやること

| 順番 | やること |
|------|----------|
| 1 | 上司に報告（電話またはメール） |
| 2 | 今日の業務を見直す（何を優先するか） |
| 3 | 応援を頼む（他の部署、他のスタッフ） |

### 2. 報告のしかた

「○○病棟です。今日の日勤が○人で、○人足りません。応援をお願いします。」

### 3. 業務の優先順位

人が足りないときの優先順位：

| 優先度 | 業務 |
|--------|------|
| 高 | 患者さんの安全、緊急の処置 |
| 中 | 定時の薬、検査の送り出し |
| 低 | 記録、整理整頓 |

### 4. 記録を残す

- いつ、何人足りなかったか
- どう対応したか
- 影響があったか

!!! warning "例外"
    - 夜勤で足りないときは、当直師長に連絡
    - 災害時は、災害対策本部の指示に従う

!!! info "相談"
    - **誰に**：看護部 人事担当
    - **いつまでに**：人が足りないとわかったらすぐ
    - **何を言う**：「○月○日、○○病棟で○人足りない。応援がほしい」

!!! danger "禁止"
    - 報告しないで何とかしようとする
    - 患者さんの安全を後回しにする
    - 1人で全部やろうとする
