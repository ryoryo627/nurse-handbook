# 書き方ルール

## まずこれだけ

ハンドブックは「中学2年生でも読める」言葉で書きます。
見出しは短く、ページは決まった型で書きます。

## できた状態

- 誰が書いても同じ形式になる
- 読む人が迷わない
- 検索で見つけやすい

## やり方

### 1. 言葉のルール

| やること | やらないこと |
|----------|--------------|
| 中学2年生の語彙で書く | ビジネス用語を並べる |
| 専門語は用語集にリンク | 説明なしで略語を使う |
| 初出だけリンクする | 同じ言葉に何度もリンク |

**例**

- ✕「ステークホルダーにアラインする」
- ○「関係者と方向を合わせる」

### 2. 見出しのルール

| やること | やらないこと |
|----------|--------------|
| 短くする | 長くする |
| 「目的」と書く | 「目的（何のため？）」と書く |
| 1つの意味で書く | 括弧で補足を入れる |

**例**

- ✕「相談（誰に聞けばいい？）」
- ○「相談」

### 3. ページの型

すべてのページはこの順番で書きます。必要ないものは省略できます。

1. **まずこれだけ**：2〜4行で要点
2. **できた状態**：ゴールの姿
3. **やり方**：手順や方法
4. **例外**：特別なケース
5. **相談**：誰に・いつまでに・何を言う
6. **禁止**：やってはいけないこと
7. **チェック**：確認リスト
8. **関連**：用語集や関連ページへのリンク

→ 正式な型は[ページテンプレート](../99_templates/page.md)を見てください

### 4. frontmatterのルール

すべてのページの先頭に書きます。

```yaml
---
title: "ページ名"
who: "読む人（看護主任・師長など）"
owner: "責任者（看護部○○担当など）"
last_checked: "2025-12-23"
---
```

## 例外

- 用語集は、この型に従わなくてよい
- テンプレートページは、型の見本なのでそのまま

## 相談

- **誰に**：看護部 教育担当
- **いつまでに**：書く前に確認したいとき
- **何を言う**：「この書き方でいいか」「どの型を使えばいいか」

## 禁止

- 見出しに絵文字を使う
- 1ページに複数のトピックを混ぜる
- 他のページをコピーして型を崩す

- [ ] ページの型に沿っているか
