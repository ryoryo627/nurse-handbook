# 使い方

## まずこれだけ

このハンドブックは「今のやり方の正解」が書いてある唯一の場所です。
やり方に迷ったら、ここを見てください。ここに書いてあることが正解です。

## できた状態

- 知りたいことがすぐに見つかる
- 書いてあることに従って動ける
- 書いてないことは相談できる

## やり方

### 1. 検索する

- 画面上の検索窓にキーワードを入れる
- 例：「会議」「引きつぎ」「苦情」

### 2. 目次から探す

- 左のメニューから章を選ぶ
- 共通OS → みんなが使うスキル
- 管理OS → 管理職の仕事
- 困ったとき → よくある困りごと

### 3. 読むときのコツ

- 「まずこれだけ」を先に読む
- 全部読まなくていい
- 必要なところだけ読む

## 例外

- 緊急時は、読んでいる暇がないので、まず動く
- あとで「正しかったか」をハンドブックで確認する

## 相談

- **誰に**：看護部 教育担当
- **いつまでに**：困ったらその日のうちに
- **何を言う**：「○○を探したけど見つからない」「○○ページの意味がわからない」

## このハンドブックの約束

1. **唯一の場所**：やり方はここに書く。他の場所には書かない
2. **先に直す**：やり方を変えるときは、先にハンドブックを直す
3. **理由を残す**：なぜ変えたかを記録する
4. **古くしない**：定期的に見直して、最新にする
