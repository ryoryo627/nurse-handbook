# 責任者一覧

## まずこれだけ

ハンドブックのページには責任者がいます。
変更するときは、そのページの責任者にレビューを頼みます。

## このページのゴール

- 誰に聞けばいいかわかる
- 変更のレビューがスムーズ
- 古くなったページがない

## 責任者の役割

1. **レビュー**：変更PRをチェックして承認する
2. **鮮度管理**：担当ページを3か月に1回は見直す
3. **質問対応**：ページの内容について聞かれたら答える

## 例外

- 責任者が不在のときは、その上司がレビューする
- 緊急の変更は、事後にレビューをもらう

## 相談

- **誰に**：看護部 教育担当
- **いつまでに**：責任者がわからないとき、すぐに
- **何を言う**：「○○ページの責任者は誰か」
