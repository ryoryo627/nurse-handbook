# 目的

!!! tip "まずこれだけ"

    このハンドブックは「今のやり方の正解」を1か所にまとめたものです。
    やり方に迷ったら、まずここを見てください。

## このページのゴール

- 主任・師長が「今のやり方」をすぐに確認できる
- 新しい主任・師長が、引きつぎなしでも動き出せる
- やり方を変えるとき、変更前と変更後がはっきりする

## やり方

1. 困ったら、まずこのハンドブックを検索する
2. 書いてあることに従う
3. 書いてないことは、上司に相談して決める
4. 決まったら、ハンドブックに追加する

!!! warning "例外"

    - 緊急時は、ハンドブックより患者さんの安全を優先
    - ハンドブックと現場が合わないときは、[変更の仕方](how-to-change.md)を見て直す

!!! info "相談"

    - **誰に**：看護部 教育担当
    - **いつまでに**：疑問に思ったら1週間以内
    - **何を言う**：「○○ページの○○がわからない」「○○が実態と違う」
