# 病棟

## まずこれだけ

病棟は24時間、患者さんを見守る場所です。
シフト制で動くので、[引きつぎ](../00_start/glossary.md#handover)が大事です。
夜勤の人員確保と、急変対応の準備が重要です。

## このページのゴール

- 24時間、必要な人員がいる
- 引きつぎで情報が正しく伝わる
- 急変にすぐ対応できる

## 病棟ならではの仕事

### 1. シフト管理

| ポイント | 内容 |
|----------|------|
| 夜勤の人数 | 最低人数を守る |
| 経験者の配置 | 各勤務帯に経験者を入れる |
| 急な欠員 | 応援体制を決めておく |

### 2. 引きつぎ

| やること | タイミング |
|----------|------------|
| 患者さんの状態 | 勤務交代時 |
| 注意すること | 勤務交代時 |
| 今日の予定 | 朝の申し送り |

### 3. 急変対応

- 急変時の役割分担を決めておく
- 救急カートの場所を全員が知っている
- 定期的に訓練する

## 例外

- 感染症の流行時は、隔離の対応を優先
- 災害時は、避難と安全確保を優先

## 相談

- **誰に**：看護部 病棟担当
- **いつまでに**：困ったときすぐに
- **何を言う**：「病棟で○○が起きている。対応を相談したい」
