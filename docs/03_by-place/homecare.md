# 在宅

!!! tip "まずこれだけ"

    在宅は、患者さんの家に行って看護する場所です。
    1人で訪問するので、判断力と準備が大事です。
    家族との連携も重要です。

## このページのゴール

- 訪問先で必要なことができる
- 家族と信頼関係ができている
- 困ったときにすぐ連絡できる

## 在宅ならではの仕事

### 1. 訪問の準備

| 準備 | 内容 |
|------|------|
| 情報確認 | 前回の記録、注意点 |
| 物品確認 | 必要な道具、薬 |
| 連絡確認 | 緊急時の連絡先 |

### 2. 訪問中

| やること | ポイント |
|----------|----------|
| 観察 | 体調、生活環境、家族の様子 |
| ケア | 計画に沿った看護 |
| 説明 | 家族にもわかるように |
| 記録 | その場でメモを取る |

### 3. 訪問後

- 記録を残す
- 次回の準備をする
- 気になることは上司に報告

!!! warning "例外"

    - 訪問先で急変があったら、まず119番
    - 危険を感じたら、その場を離れる

!!! info "相談"

    - **誰に**：直属の上司
    - **いつまでに**：訪問中に困ったらすぐ電話
    - **何を言う**：「今○○さん宅で○○が起きている。指示がほしい」
