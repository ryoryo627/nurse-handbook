# 外来

!!! tip "まずこれだけ"

    外来は、たくさんの患者さんが短時間で来る場所です。
    待ち時間の管理と、スムーズな流れが大事です。
    初めて来る患者さんにもわかりやすく案内します。

## このページのゴール

- 患者さんがスムーズに診察を受けられる
- 待ち時間が長くなりすぎない
- スタッフが効率よく動ける

## 外来ならではの仕事

### 1. 患者さんの流れ

| 場面 | やること |
|------|----------|
| 受付 | 案内、問診票の確認 |
| 待合 | 待ち時間の管理、呼び出し |
| 診察 | 医師のサポート、説明 |
| 会計 | 次回予約、帰りの案内 |

### 2. 待ち時間の管理

- 予約の間隔を調整する
- 混んでいるときは声をかける
- 「あと○分くらい」と伝える

### 3. 初めての患者さんへの対応

- 場所の案内（トイレ、自販機など）
- 書類の書き方の説明
- 困っていそうな人への声かけ

!!! warning "例外"

    - 急に具合が悪くなった患者さんは、優先して対応
    - 感染症が疑われる患者さんは、別の場所で待ってもらう

!!! info "相談"

    - **誰に**：看護部 外来担当
    - **いつまでに**：困ったときすぐに
    - **何を言う**：「外来で○○が起きている。対応を相談したい」
