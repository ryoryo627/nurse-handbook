# 1対1

## まずこれだけ

[1対1](../00_start/glossary.md#one-on-one)は、上司と部下が2人だけで話す時間です。
月に1回、15〜30分を目安にします。
スタッフの話を聞くことが目的です。指示を出す場ではありません。

## できた状態

- スタッフが困っていることを早く知れる
- スタッフとの信頼関係ができる
- 問題が大きくなる前に対処できる

## やり方

### 1. 準備

| やること | 例 |
|----------|-----|
| 日時を決める | 月1回、15〜30分 |
| 場所を決める | 個室、または静かな場所 |
| 前回の話を見返す | 前回困っていたことは解決したか |

### 2. 話すこと

スタッフに話してもらう。こちらが話しすぎない。

| 質問の例 |
|----------|
| 「最近どう？」 |
| 「困っていることある？」 |
| 「前に話してた○○は、その後どう？」 |
| 「何か手伝えることある？」 |

### 3. 聞き方

| やること | やらないこと |
|----------|--------------|
| うなずく、相づち | すぐにアドバイスする |
| 最後まで聞く | 途中で遮る |
| メモを取る | スマホを見る |
| 「それで？」と促す | 「でも」「だけど」と否定 |

### 4. 終わり方

- 「今日話したこと、覚えておくね」と伝える
- 次回の日程を決める
- 必要なことがあれば、あとで対応する

## 例外

- スタッフが話したくなさそうなら、無理に聞かない
- 「また話したくなったら声かけて」と伝える

## 相談

- **誰に**：直属の上司
- **いつまでに**：スタッフから深刻な話を聞いたとき、その日のうち
- **何を言う**：「○○さんから○○という話があった。対応を相談したい」

## 禁止

- 1対1の場で評価や叱責をする
- 他の人に1対1の内容を話す（守秘義務）
- 約束した1対1をキャンセルし続ける

## チェック

- [ ] 月に1回できているか
- [ ] スタッフが話す時間が多かったか
- [ ] メモを残したか
- [ ] 次回の日程を決めたか
