# 管理OS

## まずこれだけ

管理OSは、看護管理職が使う専門スキルです。
人の管理、問題対応、数字の把握など、管理職ならではの仕事をまとめています。

## スキル一覧

| スキル | 何をする | ページ |
|--------|----------|--------|
| 人員調整 | シフトを組む、人を配置する | [人員調整](staffing.md) |
| 1対1 | スタッフと個別に話す | [1対1](one-on-one.md) |
| 教える | 新人や後輩を育てる | [教える](teaching.md) |
| 事故報告 | 事故が起きたときの対応 | [事故報告](incident.md) |
| 苦情対応 | 苦情が来たときの対応 | [苦情対応](complaints.md) |
| 数字 | 数字を見て判断する | [数字](numbers.md) |
