# 数字

## まずこれだけ

数字を見ることで、問題に早く気づけます。
毎月見る数字を決めて、変化を追います。
数字が悪いときは、原因を考えて対策します。

## このページのゴール

- 毎月見る数字が決まっている
- 数字の変化に気づける
- 数字をもとに次の行動を決められる

## やり方

### 1. 見るべき数字

| 分類 | 数字の例 | 見方 |
|------|----------|------|
| 人 | 離職率、残業時間 | 増えていないか |
| 安全 | 事故件数、転倒件数 | 減っているか |
| 業務 | 病床稼働率、在院日数 | 目標に近いか |
| 患者 | 苦情件数、満足度 | 悪化していないか |

### 2. 数字の見方

1. **前月と比べる**
   - 増えたか、減ったか
   - なぜ変わったか

2. **目標と比べる**
   - 目標に達しているか
   - 届いていないなら、何をするか

3. **傾向を見る**
   - 3か月、半年のグラフを見る
   - 上がっているか、下がっているか

### 3. 数字を使う

| 場面 | 使い方 |
|------|--------|
| 会議で共有 | 「先月の残業は○時間でした」 |
| 上司に報告 | 「離職率が上がっています。対策を相談したい」 |
| 改善を考える | 「事故が多い時間帯は○時台。対策は…」 |

## 例外

- 数字だけで判断しない
- 現場の声や状況も合わせて考える

## 相談

- **誰に**：直属の上司
- **いつまでに**：数字が急に悪くなったとき
- **何を言う**：「○○の数字が悪化している。原因を一緒に考えたい」
