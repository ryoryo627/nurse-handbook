# 苦情対応

## まずこれだけ

[苦情](../00_start/glossary.md#complaint)は「困っている」というサインです。
まず話を聞く。言い訳をしない。
対応したことを記録に残します。

## このページのゴール

- 相手が「話を聞いてもらえた」と感じている
- 問題の原因がわかっている
- 対応したことが記録に残っている

## やり方

### 1. まず聞く

| やること | やらないこと |
|----------|--------------|
| 最後まで聞く | 途中で遮る |
| 「申し訳ありません」と言う | 言い訳をする |
| メモを取る | 腕を組む |
| 「それは大変でしたね」と共感 | 「でも」「だって」と言う |

### 2. 内容を確認する

- 「○○ということですね」と復唱する
- 何が問題だったか整理する
- 「確認してお返事します」と伝える

### 3. 対応を決める

| ケース | 対応 |
|--------|------|
| その場で解決できる | 説明して、お詫びする |
| 確認が必要 | 「○日までに連絡します」と伝える |
| 自分で判断できない | 上司に相談して、一緒に対応する |

### 4. 記録を残す

- いつ、誰から、何があったか
- どう対応したか
- 結果はどうだったか

## 例外

- 暴言・暴力がある場合は、その場を離れて上司を呼ぶ
- 無理な要求には「できません」と伝えてよい

## 相談

- **誰に**：患者相談室、または直属の上司
- **いつまでに**：苦情を受けたらその日のうち
- **何を言う**：「○○さんから苦情があった。対応を相談したい」

## 禁止

- 「私じゃわかりません」と突き放す
- たらい回しにする
- 記録しないで終わる
- 1人で抱え込む
