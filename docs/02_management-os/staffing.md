# 人員調整

## まずこれだけ

人員調整は、必要な人を必要な時間に配置することです。
シフトを組むだけでなく、急な欠員への対応も含みます。
「患者さんの安全」と「スタッフの健康」の両方を考えます。

## このページのゴール

- 各勤務帯に必要な人数がいる
- スタッフの希望がなるべく反映されている
- 急な欠員にも対応できる

## やり方

### 1. シフトを組む

| ステップ | やること |
|----------|----------|
| 1 | 必要人数を確認（日勤○人、夜勤○人） |
| 2 | スタッフの希望を集める |
| 3 | 経験者と新人のバランスを考える |
| 4 | 案を作って確認してもらう |
| 5 | 確定して周知する |

### 2. 急な欠員への対応

1. まず代わりを探す（同じ部署の人）
2. いなければ応援を頼む（他の部署）
3. それでも無理なら上司に相談

### 3. 人が足りないとき

- [人が足りない](../10_troubles/not-enough-people.md)を見る
- 上司に早めに伝える
- 記録を残す（いつ、何人足りなかったか）

## 例外

- 災害時は、通常のシフトを変更して対応
- 感染症の流行時は、隔離要員を優先

## 相談

- **誰に**：看護部 人事担当
- **いつまでに**：シフト確定の1週間前
- **何を言う**：「来月は○日が人が足りない」「応援がほしい」
