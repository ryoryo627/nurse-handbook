# 事故報告

## まずこれだけ

[事故報告](../00_start/glossary.md#incident-report)は、起きたことを正しく記録して、再発を防ぐためのものです。
「報告すること」が目的ではなく「次に同じことを起こさない」が目的です。
隠さない、責めない、すぐに報告する。

## できた状態

- 何が起きたか正しく記録されている
- 患者さんへの対応が終わっている
- 再発防止のために何をするか決まっている

## やり方

### 1. 事故が起きたら

| 順番 | やること |
|------|----------|
| 1 | 患者さんの安全を確保する |
| 2 | 上司に口頭で報告する |
| 3 | 患者さん・家族に説明する |
| 4 | 報告書を書く |
| 5 | 再発防止を考える |

### 2. 報告書の書き方

| 項目 | 書くこと |
|------|----------|
| いつ | 日時（○月○日○時○分） |
| どこで | 場所（○病棟○号室など） |
| 誰が | 当事者 |
| 何が | 何が起きたか（事実のみ） |
| どうした | どう対応したか |
| 今後 | 再発防止のために何をするか |

### 3. 上司としての対応

- 報告してくれたことに「ありがとう」と言う
- 責めない（責めると次から報告が来なくなる）
- 一緒に再発防止を考える
- 必要なら看護部長に報告する

## 例外

- 重大な事故（患者さんが亡くなった、大きな障害が残った）は、すぐに看護部長に報告
- 報告書より先に口頭で伝える

## 相談

- **誰に**：看護部 安全担当
- **いつまでに**：事故が起きたら24時間以内
- **何を言う**：「○○で事故が起きた。報告書は○時までに出す」

## 禁止

- 報告を遅らせる
- 事実を変える
- 当事者を責める
- 「報告しなくていい」と言う

## チェック

- [ ] 患者さんの安全は確保したか
- [ ] 上司に口頭で報告したか
- [ ] 患者さん・家族に説明したか
- [ ] 報告書を24時間以内に書いたか
- [ ] 再発防止を考えたか
