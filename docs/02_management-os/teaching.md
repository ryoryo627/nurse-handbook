# 教える

!!! tip "まずこれだけ"

    教えるとは、相手ができるようになることです。
    説明しただけでは教えたことになりません。
    「やってみせる→やらせる→ふりかえる」の順番で進めます。

## このページのゴール

- 相手が1人でできるようになっている
- 相手が「わからないときに聞ける」と思っている
- 相手が自信を持っている

## やり方

### 1. 教える前

| やること | 例 |
|----------|-----|
| ゴールを決める | 「○○が1人でできる」 |
| 相手のレベルを知る | 経験はあるか、何が苦手か |
| 時間を確保する | 忙しい中で教えない |

### 2. 教える順番

1. **やってみせる**
   - 自分がやって見せる
   - ポイントを説明する
   - 「ここ見てね」と注目点を伝える

2. **やらせる**
   - 相手にやってもらう
   - そばで見守る
   - 危険なときだけ止める

3. **ふりかえる**
   - 「どうだった？」と聞く
   - よかった点を伝える
   - 次に気をつける点を1つだけ伝える

### 3. くりかえす

- 1回でできなくて当たり前
- できるまで何度も繰り返す
- 少しずつ難しいことに挑戦

!!! warning "例外"

    - 緊急時は、説明なしで指示を出してよい
    - あとで「なぜああしたか」を説明する

!!! info "相談"

    - **誰に**：看護部 教育担当
    - **いつまでに**：教え方に困ったとき
    - **何を言う**：「○○さんに○○を教えているが、うまくいかない」

!!! danger "禁止"

    - 「前にも言った」と責める
    - 人前で失敗を指摘する
    - 「見て覚えろ」で終わらせる
