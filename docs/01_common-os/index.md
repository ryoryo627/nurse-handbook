# 仕事の基本

## まずこれだけ

仕事の基本は、看護管理職が毎日使う5つの基本スキルです。
どの病棟でも、どの役職でも使います。

## 5つのスキル

| スキル | 何をする | ページ |
|--------|----------|--------|
| 言葉にする | 考えを言葉にして整理する | [言葉にする](language.md) |
| 伝える | 相手にわかるように伝える | [伝える](tell.md) |
| 任せる | 仕事を人に任せる | [任せる](delegate.md) |
| 会議 | 会議を進める | [会議](meeting.md) |
| 記録 | 決まったことを残す | [記録](notes.md) |

## どこから読む？

- 全部読む必要はありません
- 困っているスキルから読んでください
- 「まずこれだけ」を先に読むと、要点がわかります
