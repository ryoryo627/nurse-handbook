# 記録

!!! tip "まずこれだけ"

    記録は「あとで誰かが見る」ために残します。
    自分だけわかるメモではなく、他の人が読んでわかる形で書きます。
    最低限「いつ・何が決まった・誰がやる」を書きます。

## このページのゴール

- あとから見て、何があったかわかる
- 誰が読んでも同じ理解になる
- 必要なときにすぐ見つかる

## やり方

### 1. 何を記録するか

必ず記録すること：

| 場面 | 記録すること |
|------|--------------|
| 会議 | 決まったこと、誰がやるか、いつまで |
| [引きつぎ](../00_start/glossary.md#handover) | 今の状態、注意すること、次にやること |
| 相談 | 何を相談したか、どう答えたか |
| 問題 | 何が起きたか、どう対応したか |

### 2. どう書くか

| やること | やらないこと |
|----------|--------------|
| 箇条書きで短く | 長い文章を書く |
| 事実を書く | 感想だけ書く |
| 日付を入れる | 日付なしで書く |
| 名前を入れる | 「誰か」とあいまいに書く |

**例**

```
2025-12-23 シフト調整会議

決まったこと
- 来月から夜勤の人数を4人→3人に変更
- 1月15日から開始

やること
- 山田：スタッフへ説明（12/25まで）
- 佐藤：シフト表の修正（12/27まで）
```

### 3. どこに残すか

| 記録の種類 | 保存場所 |
|------------|----------|
| 会議メモ | 共有フォルダの「会議」 |
| 決まったこと | ハンドブックまたは決定ログ |
| 患者に関すること | 電子カルテ |
| 事故・苦情 | 報告システム |

!!! warning "例外"

    - 緊急時は、口頭で伝えて、あとから記録する
    - 「30分以内に書く」を目安にする

!!! info "相談"

    - **誰に**：直属の上司
    - **いつまでに**：「これは記録すべきか」と思ったとき
    - **何を言う**：「○○を記録したいが、どこに残すべきか」

!!! danger "禁止"

    - 記録しないで終わる
    - 自分だけがわかる書き方をする
    - 古い記録を捨てる（保管期間を確認）
