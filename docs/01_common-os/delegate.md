# 任せる

## まずこれだけ

任せるとは、仕事を渡して、結果に責任を持つことです。
「丸投げ」ではありません。
任せたあとも、進み具合を確認します。

## このページのゴール

- 相手が何をすればいいかわかっている
- 期限と求める結果がはっきりしている
- 困ったときに相談できる関係がある

## やり方

### 1. 何を任せるか決める

任せていいこと、自分でやるべきことを分ける。

| 任せていい | 自分でやる |
|------------|------------|
| 決まった手順の仕事 | 最終判断 |
| 相手が得意なこと | 人事に関わること |
| 成長につながること | お金に関わること |

### 2. 伝えること

任せるときに、これを伝える。

| 項目 | 例 |
|------|-----|
| 何を | 「来月のシフト案を作って」 |
| いつまで | 「今週金曜まで」 |
| どこまで | 「案を作るところまで。決定は私がする」 |
| 相談のタイミング | 「困ったら水曜までに言って」 |

### 3. 途中で確認する

- 任せっぱなしにしない
- 中間で「どう？」と聞く
- 問題があれば早めに気づける

### 4. 結果を一緒に見る

- 終わったら結果を確認する
- よかった点、次に活かす点を伝える
- 「ありがとう」を言う

## 例外

- 緊急時は、細かく伝える余裕がない
  - まず「○○して」と指示
  - あとで「なぜそうしたか」を説明
- 初めての仕事は、最初は細かく確認
  - 慣れてきたら確認の間隔を広げる

## 相談

- **誰に**：自分の上司
- **いつまでに**：「これを任せていいか」と思ったとき
- **何を言う**：「○○さんに○○を任せたいが、よいか」

## 禁止

- 任せたのに細かく口を出す
- 任せたことを忘れて放置する
- 失敗したときに「言ったはず」と責める
