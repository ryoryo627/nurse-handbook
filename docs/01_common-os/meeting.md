# 会議

!!! tip "まずこれだけ"

    会議は「決める場」か「共有する場」です。
    目的がないなら、会議をしない。
    終わったら「何が決まったか」を残します。

## このページのゴール

- 参加者が「何のための会議か」わかっている
- 決めるべきことが決まっている
- 決まったことが記録に残っている

## やり方

### 1. 会議の前

| やること | 例 |
|----------|-----|
| 目的を決める | 「シフトの調整方法を決める」 |
| 議題を決める | 「案A・案Bのどちらにするか」 |
| 参加者を決める | 決定に必要な人だけ呼ぶ |
| 時間を決める | 30分、長くても1時間 |
| 資料を配る | 前日までに送る |

### 2. 会議の進め方

1. **最初に**
   - 「今日の目的は○○です」と言う
   - 「○時に終わります」と言う

2. **議論**
   - 1つずつ話す
   - 脱線したら「それは別で」と止める
   - 意見が出ないときは指名する

3. **最後に**
   - 「決まったことは○○です」と確認
   - 「次にやることは○○です」と確認
   - 「誰がやるか」を決める

### 3. 会議のあと

- 決まったことをメモにして共有する
- 次回の日程を決める（必要なら）
- やることを始める

!!! warning "例外"

    - 緊急の集まりは、議題なしで始めてよい
      - 終わったあとに「何が決まったか」を記録する
    - 情報共有だけなら、メールで済むかも考える

!!! info "相談"

    - **誰に**：会議を開きたい相手の上司
    - **いつまでに**：会議の1週間前
    - **何を言う**：「○○について会議をしたい。目的は○○」

!!! danger "禁止"

    - 目的なく集まる
    - 決まったことを記録しない
    - 参加者が多すぎる（決める会議は5人以下）
