# 言葉にする

## まずこれだけ

頭の中のモヤモヤを、言葉にして出すことが第一歩です。
言葉にしないと、自分も相手もわかりません。
まず短い言葉で言ってみる。それから詳しくする。

## このページのゴール

- 自分が何を考えているか、自分でわかる
- 相手に伝える準備ができる
- 問題が整理されて、次にやることが見える

## やり方

### 1. まず一言で言ってみる

- 「何が問題？」と聞かれたら、一言で答える
- 例：「人が足りない」「報告が遅い」「方針がわからない」

### 2. 5W1Hで広げる

一言で言えたら、詳しくする。

| 項目 | 質問 | 例 |
|------|------|-----|
| What | 何が問題？ | 夜勤の人が足りない |
| Who | 誰が関係？ | 病棟のスタッフ全員 |
| When | いつから？いつまで？ | 来月から3か月 |
| Where | どこで？ | A病棟 |
| Why | なぜ起きた？ | 2人が産休に入る |
| How | どうする？ | 応援を頼む |

### 3. 書いて確認する

- 言葉にしたら、紙かメモに書く
- 書くと、抜けやズレに気づく
- 書いたものを見せて確認してもらう

## 例外

- 緊急時は、書くより先に口で伝える
- あとで書いて記録に残す

## 相談

- **誰に**：直属の上司、または先輩の主任・師長
- **いつまでに**：モヤモヤが2日続いたら
- **何を言う**：「こういうことで困っている。整理を手伝ってほしい」
