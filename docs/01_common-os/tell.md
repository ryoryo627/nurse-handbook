# 伝える

## まずこれだけ

伝えるとは、相手が「わかった」と思う状態にすることです。
話しただけでは伝わっていません。
相手の反応を見て、伝わったか確認します。

## できた状態

- 相手が内容を理解している
- 相手が次に何をすればいいかわかっている
- 誤解がない

## やり方

### 1. 結論から言う

- 最初に「何をしてほしいか」「何が言いたいか」を言う
- 理由や背景はあとから

**例**

- ✕「昨日の夜勤で○○があって、△△さんが困っていて、それで…」
- ○「来週の会議で、○○について話し合いたいです。理由は…」

### 2. 短く区切る

- 一度にたくさん言わない
- 1つ伝えたら、相手の反応を見る
- 「ここまでで質問ある？」と聞く

### 3. 確認する

伝えたあと、相手に確認する。

| やり方 | 例 |
|--------|-----|
| 復唱してもらう | 「今の内容を言ってみて」 |
| 質問する | 「何から始める？」 |
| 反応を見る | うなずいているか、困った顔か |

### 4. 伝える手段を選ぶ

| 手段 | 使う場面 |
|------|----------|
| 口頭 | 急ぎ、感情が関わる、すぐ確認したい |
| メール | 記録を残したい、あとで見返したい |
| 文書 | 正式に残す、複数人に同じ内容を伝える |
| 会議 | 全員で共有、意見を聞きたい |

## 例外

- 緊急時は、確認を省いてまず動いてもらう
- あとで「さっきの件、大丈夫だった？」と確認する

## 相談

- **誰に**：伝えたい相手の上司、または自分の上司
- **いつまでに**：伝えにくい内容なら、伝える前に
- **何を言う**：「○○さんに○○を伝えたいが、どう言えばいいか」

## 禁止

- 「言ったはず」で終わらせる
- 相手が忙しいときに長く話す
- メールだけで済ませて、反応を見ない
