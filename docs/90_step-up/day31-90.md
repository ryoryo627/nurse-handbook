# 着任31〜90日

## まずこれだけ

31〜90日は「動く」時期です。
小さなことから改善を始めます。
スタッフと一緒に考える姿勢を見せます。

## できた状態

- 日々の業務を1人で回せる
- 小さな改善を1つ以上始めている
- スタッフとの信頼関係ができ始めている

## やること

### 31〜45日：自分で回す

| やること | ポイント |
|----------|----------|
| 日々の業務を自分で判断 | 迷ったら相談 |
| シフト調整をやってみる | 最初は確認してもらう |
| 会議を進行してみる | 準備をしっかり |

### 46〜60日：課題を見つける

| やること | ポイント |
|----------|----------|
| 困っていることを整理 | 自分が感じたこと |
| スタッフに聞く | 「何が困っている？」 |
| 数字を見る | 残業、事故、苦情など |

### 61〜90日：小さく始める

| やること | ポイント |
|----------|----------|
| 改善案を考える | 小さなことから |
| スタッフと相談する | 「こうしたらどうか」 |
| 上司に報告する | 「○○を変えたい」 |
| 実行する | まずやってみる |

## 例外

- 大きな問題があれば、上司と相談して対応
- 無理に変えようとしない

## 相談

- **誰に**：直属の上司
- **いつまでに**：月に1回は状況報告
- **何を言う**：「こんな課題がある」「こうしたいと思う」

## 禁止

- 一度にたくさん変えようとする
- スタッフに相談せず決める
- 失敗を恐れて何もしない
