# 着任91日〜1年

## まずこれだけ

91日以降は「広げる」時期です。
チームをまとめて、目標に向かって進みます。
自分だけでなく、スタッフが育つ環境を作ります。

## このページのゴール

- チームとして動けている
- 目標に向かって進んでいる
- スタッフが成長している

## やること

### 91〜180日：チームをまとめる

| やること | ポイント |
|----------|----------|
| 目標を共有する | みんなで同じ方向を向く |
| 役割を任せる | スタッフに仕事を任せる |
| 定期的に振り返る | 月に1回は全体を見る |

### 181〜270日：スタッフを育てる

| やること | ポイント |
|----------|----------|
| 1対1を続ける | 月に1回はスタッフと話す |
| 教える機会を作る | 勉強会、研修 |
| 任せて見守る | 失敗も経験 |

### 271〜365日：1年を振り返る

| やること | ポイント |
|----------|----------|
| 数字を見る | 目標は達成したか |
| 変わったことを整理する | 良くなったこと、課題 |
| 次の目標を考える | 2年目に向けて |

## 例外

- 大きな問題があれば、すべてを止めて対応
- 自分だけで抱えない

## 相談

- **誰に**：直属の上司
- **いつまでに**：四半期に1回は振り返り
- **何を言う**：「この3か月でこうなった」「次はこうしたい」
