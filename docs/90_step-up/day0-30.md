# 着任0〜30日

## まずこれだけ

最初の30日は「知る」ことに集中します。
大きな変更は避けて、まず現状を理解します。
困ったらすぐに上司に相談します。

## できた状態

- スタッフの名前と顔が一致する
- 病棟・部署の流れがわかる
- 誰に何を相談すればいいかわかる

## やること

### 1週目：顔を覚える

| やること | ポイント |
|----------|----------|
| スタッフに挨拶する | 名前を覚える |
| 一緒に動いてみる | 流れを見る |
| 質問する | 「いつも○○はどうしてますか？」 |

### 2週目：流れを知る

| やること | ポイント |
|----------|----------|
| 1日の流れを確認 | 朝から夜まで |
| 会議に出る | どんな会議があるか |
| 書類を見る | どんな記録があるか |

### 3〜4週目：関係を作る

| やること | ポイント |
|----------|----------|
| 1対1で話す | スタッフと少しずつ |
| 他部署に挨拶 | 医師、薬剤師、事務など |
| 上司に報告 | 気づいたこと、困ったこと |

## 例外

- 緊急事態があれば、上司の指示で対応
- 「前のやり方」を否定しない

## 相談

- **誰に**：直属の上司
- **いつまでに**：困ったらすぐ
- **何を言う**：「○○がわからない」「○○について相談したい」

## 禁止

- 着任早々「前の職場では…」と比較する
- いきなり大きなルールを変える
- 1人で問題を解決しようとする
