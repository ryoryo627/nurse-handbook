# 会議テンプレート

会議のメモを取るときのひな形です。
コピーして使ってください。

---

## ひな形

```markdown
# ○○会議

## 基本情報

| 項目 | 内容 |
|------|------|
| 日時 | 2025年○月○日（○）○:○〜○:○ |
| 場所 | ○○会議室 |
| 参加者 | ○○、○○、○○ |
| 記録 | ○○ |

## 目的

今日の会議で何を決めるか、何を共有するか

## 議題

### 1. ○○について

**背景**
- なぜこの話をするか

**話し合ったこと**
- ○○という意見があった
- △△という意見もあった

**決まったこと**
- ○○することに決まった

**やること**
| 誰が | 何を | いつまで |
|------|------|----------|
| ○○ | △△ | ○月○日 |

### 2. ○○について

（同じ形式で書く）

## 次回

| 項目 | 内容 |
|------|------|
| 日時 | 2025年○月○日（○）○:○〜 |
| 議題 | ○○ |
```

---

## 書き方のポイント

1. **決まったことを必ず書く**
   - 「話し合った」だけで終わらない
   - 何がどう決まったかを明確に

2. **やることを具体的に**
   - 誰が
   - 何を
   - いつまでに

3. **その日のうちに共有**
   - 参加者に送る
   - 必要なら欠席者にも
