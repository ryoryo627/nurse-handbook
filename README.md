# 看護管理職ハンドブック

看護主任・師長のための実践ガイドです。

## 目的

- 「今のやり方」を1か所にまとめる
- 新しい主任・師長がすぐに動けるようにする
- やり方を変えるときの土台にする

## ローカルで見る

```bash
# 1. Pythonが必要です（3.8以上）
python --version

# 2. 必要なものをインストール
pip install -r requirements.txt

# 3. ローカルサーバーを起動
mkdocs serve

# 4. ブラウザで開く
# http://127.0.0.1:8000
```

## 編集ルール

1. **直すときはPRで**
   - ブランチを作る
   - 変更する
   - PRを出す
   - レビューを受ける
   - マージする

2. **難しい言葉は用語集へ**
   - 専門用語や略語は `docs/00_start/glossary.md` に追加
   - 本文では用語集へリンクする

3. **見出しは短く**
   - 「目的（何のため？）」→「目的」
   - 括弧の補足は原則使わない

4. **last_checked を更新**
   - ページを直したら、frontmatterの `last_checked` を今日に

## デプロイ

- `master` ブランチにマージすると自動でGitHub Pagesに公開
- GitHub Actions が `mkdocs build` を実行
- 数分で反映される

## ファイル構成

```
docs/
  00_start/     # はじめる前に（使い方、ルール）
  01_common-os/ # 共通OS（みんなが使うスキル）
  02_management-os/ # 管理OS（管理職の仕事）
  03_by-place/  # 場所別（病棟、外来、在宅）
  10_troubles/  # 困ったとき
  90_step-up/   # ステップアップ
  99_templates/ # テンプレート
```
