# 見た目凍結ルール

このリポジトリの見た目（CSS/テーマ/HTML）は凍結されています。

## 凍結対象ファイル

このリポジトリで見た目に関わるファイル:

| ファイル | 内容 | 状態 |
|----------|------|------|
| `mkdocs.yml` | テーマ設定（Material, teal配色） | 凍結 |
| `.github/workflows/deploy.yml` | デプロイ設定 | 凍結 |

現在、カスタム CSS やオーバーライドは存在しません。

## 原則

- 上記ファイルは原則として変更しない
- テーマの色（primary: teal, accent: teal）を変えない
- 新しい CSS やオーバーライドを追加しない

## 変更が必要な場合

どうしても見た目を変える必要がある場合:

1. **理由を明記** - なぜ変更が必要か
2. **影響範囲を確認** - 何が変わるか
3. **変更前後を比較** - スクリーンショットで確認
4. **ロールバック手順を用意** - 戻せるようにする
5. **文章変更PRとは分ける** - 見た目変更は別PRで

## 禁止

- 文章変更PRに見た目変更を混ぜる
- 理由なく色やフォントを変える
- デプロイを壊す変更をする
