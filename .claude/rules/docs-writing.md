# 文章スタイルルール

docs/ 配下の Markdown ファイルに適用。

## 中2語彙ルール

| やること | やらないこと |
|----------|--------------|
| 中学2年生の語彙で書く | ビジネス用語を並べる |
| 専門語は用語集にリンク | 説明なしで略語を使う |
| 初出だけリンクする | 同じ言葉に何度もリンク |

**悪い例**: 「ステークホルダーにアラインする」
**良い例**: 「関係者と方向を合わせる」

## 見出しルール

| やること | やらないこと |
|----------|--------------|
| 短くする | 長くする |
| 「目的」と書く | 「目的（何のため？）」と書く |
| 1つの意味で書く | 括弧で補足を入れる |

**悪い例**: 「相談（誰に聞けばいい？）」
**良い例**: 「相談」

## 1文ルール

- 1文を短くする
- 箇条書き中心で書く
- 長い説明は表を使う

## 「相談」セクションの書き方

必ず3点を書く:

- **誰に**: 相談先（看護部○○担当など）
- **いつまでに**: いつまでに相談するか
- **何を言う**: 何を伝えればいいか

## 禁止

- 見出しに絵文字を使う
- 1ページに複数のトピックを混ぜる
- 他のページをコピーして型を崩す
