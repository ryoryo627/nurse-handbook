# 文章の書き方ルール

## 中学2年生語彙ルール

| やること | やらないこと |
|----------|--------------|
| 中学2年生でもわかる言葉 | ビジネス用語を並べる |
| 専門語は用語集にリンク | 説明なしで略語を使う |
| 初出だけリンクする | 同じ言葉に何度もリンク |

### NG → OK 例

- NG: 「ステークホルダーにアラインする」
- OK: 「関係者と方向を合わせる」

## 見出しルール

| やること | やらないこと |
|----------|--------------|
| 短くする | 長くする |
| 「目的」と書く | 「目的（何のため？）」と書く |
| 1つの意味で書く | 括弧で補足を入れる |

### NG → OK 例

- NG: 「相談（誰に聞けばいい？）」
- OK: 「相談」

## 文章ルール

- 1文は短く
- 箇条書きを使う
- 絵文字は見出しに使わない

## 「相談」セクションの書き方

必ず3点を書く：

- **誰に**: 相談先
- **いつまでに**: 期限
- **何を言う**: 伝える内容

## 用語集リンク

- 用語集: `docs/00_start/glossary.md`
- リンク形式: `[用語名](../00_start/glossary.md#anchor)`
- **初出のみリンク**（同じページで2回目以降はリンクしない）
