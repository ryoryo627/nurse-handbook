# 共通ルール

このリポジトリは**看護管理職ハンドブック**（MkDocs Material + GitHub Pages）。

## 基本方針

- **SSoT**: ハンドブックが「今のやり方」の唯一の正解
- **現場とドキュメントを一致させる**: 変えるなら先にドキュメントを直す
- **壊さない**: 見た目、ビルド、既存ページを壊さない

## ファイル構成

```
docs/
  00_start/       # はじめる前に
  01_common-os/   # 仕事の基本
  02_management-os/ # 管理の基本
  03_by-place/    # 場所別
  10_troubles/    # 困ったとき
  90_step-up/     # ステップアップ
  99_templates/   # テンプレート
  stylesheets/    # CSS（変更禁止）
overrides/        # テーマ上書き（変更禁止）
mkdocs.yml        # 設定（慎重に）
```

## 作業前のチェック

1. 変更対象を確認（docs配下か、設定か）
2. 該当するルールファイルを確認
3. テンプレートを参照
