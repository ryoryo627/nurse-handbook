# 共通ルール

このリポジトリは「看護管理職ハンドブック」の MkDocs ドキュメントです。

## 最優先事項

1. **見た目を変えない** - mkdocs.yml のテーマ設定を維持
2. **文章トーンを維持** - 中2語彙（中学2年生が読める言葉）
3. **ハンドブック優先** - やり方を変えるなら先にドキュメントを直す
4. **SSoT** - 同じルールを複数の場所に書かない

## 禁止事項

- テーマの色（teal）を変える
- 見出しに絵文字を使う
- ビジネス用語を増やす
- 現場だけ変えてドキュメントを放置

## 確認コマンド

```bash
mkdocs build    # ビルドが通ることを確認
```
